Scan this QR code on your phone to open BotFather in Telegram:

Or click here to open on this device

In BotFather, send these messages:
1. Send this command:
/newbot
2. When asked for a name, type:
Minder Home
3. When asked for a username, type something unique:
minder_home_bot
Must end in "bot". Add numbers if taken.
4. BotFather will reply with a token that looks like:
123456789:ABCdefGHIjklMNOpqrSTUvwxYZ
5. Then disable privacy mode so the bot can hear @mentions in your group:
/setprivacy
Pick your bot, choose Disable. Do this before adding the bot to the group.
Set up a Discord bot (works where Telegram is blocked):
1. Open the Discord Developer Portal → New Application.
2. Bot → enable Message Content Intent, then Reset & copy the bot Token.
3. OAuth2 → URL Generator → scope bot, perms Send Messages + Attach Files → open the generated URL to invite it to your server.
4. Paste the token below, then @mention the bot in the channel Minder should use.
